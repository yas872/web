Sobre o projeto

Este repositório foi desenvolvido como parte de uma atividade prática com o objetivo de aprender e aplicar os conceitos fundamentais de desenvolvimento web utilizando HTML e CSS.
O projeto aborda desde a estrutura básica de páginas até a criação de formulários, tabelas e estilização com CSS externo, além da publicação utilizando o GitHub Pages.

O projeto é composto pelos seguintes arquivos:

1.html → Estrutura básica HTML + explicação sobre Internet vs Web + história com elementos diversos
2_http.html → Explicação do protocolo HTTP com exemplos de requisição e resposta
3_tabela_uf.html → Tabela com estados brasileiros organizados por região
4_curriculo.html → Página de currículo com navegação interna, tabela e formatação
4_contato.html → Formulário de contato com validações HTML
style.css → Arquivo de estilização aplicado às páginas

Estilização (CSS)

O arquivo style.css foi utilizado para:

Definir cores de fundo e texto
Padronizar fontes
Estilizar tabelas
Melhorar a aparência dos formulários
Aplicar efeitos de interação

Este projeto foi importante para reforçar conceitos básicos de desenvolvimento web, principalmente na organização de arquivos, uso correto de tags HTML e integração com CSS.
Também foi uma ótima introdução ao uso do Git e GitHub para versionamento e publicação de projetos.
