Internet, web e suas diferenças

A internet é a infraestrutura física global (cabos, servidores, roteadores) que conecta computadores, funcionando como uma "estrada". A web (World Wide Web), por outro lado, é um sistema de informações e páginas (sites) que roda sobre essa infraestrutura, agindo como o "veículo" ou conteúdo que você acessa via navegador.

Principais marcos históricos:

  • 1969 - Criação da ARPANET
  • 1983 - Surgimento do protocolo TCP/IP
  • 1989 - Tim Berners-Lee cria a Web
  • 1991 - Primeiro site publicado
