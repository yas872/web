Protocolo HTTP

O HTTP ou Protocolo de Transferência de Hipertexto é o conjunto de regras fundamental para a troca de dados na web. Ele permite que clientes (como navegadores) enviem requisições para servidores, que retornam recursos como páginas HTML, imagens e dados.

Exemplo de Requisição

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Chrome

Campos do Request

  • GET: método
  • Host: servidor
  • User-Agent: navegador

Exemplo de Response

HTTP/1.1 200 OK
Content-Type: text/html
